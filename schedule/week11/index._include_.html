Questions to discuss during tutorial:

While the tutor is checking team/individual milestones, divide these questions among team members and get ready to answer them.

  1. How does an OO Domain Model differ from a class diagram?

  2. Draw an OODM for the description below, about how modules work in a certain university:

    A module has a name and a code. A module is taken by 10 or more students, and taught by one instructor. A module can have a number tasks which can be assignments or tests. Some assignments are compulsory. When a student attempts a task, a grade and an optional feedback is given.

  3. Apply heuristics for combining multiple test inputs to the following test cases, assuming all 6 values in the table need to be tested. ✅ indicates valid values and ❗️ indicates invalid values. Point out where the heuristics are contradicted and how to improve the test cases.

    SUT: foo(x,y)

    Test case x y
    TC1 ✅ x1-valid ✅ y1-valid
    TC2 ✅ x2-valid ❗️ y2-invalid
    TC3 ❗️ x3-invalid ❗️ y3-invalid

  4. Distinguish between sequential and iterative processes. Which one are we using in the project?

  5. Distinguish between breadth-first and depth-first iterative processes using your project as an example.

  6. Describe how agile process models differ from traditional process models.

  7. Describe some distinguishing features of Scrum. Relate those features to your project.  i.e. are they applicable? are you doing something similar?

  8. When do you use MVC pattern? Do we have MVC in AB4?

  9. When do you use the Observer pattern? Do we have the Observer pattern in AB4?
