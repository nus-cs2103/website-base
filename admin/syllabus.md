## Learning Outcomes

* Able to conceptualize a software product that satisfies user requirements for a problem domain of moderate complexity.{ texts="['O1.', 'O2.', 'O3.', 'O4.', 'O5.']" }
* Able to use the object-oriented approach to design a software product of moderate complexity.
* Able to use appropriate tools (e.g. IDEs, Version Control Software, Unit Testing Frameworks) to iteratively implement, evolve, test, and deploy a software product of moderate complexity.
* Able to document, explain, and demonstrate the external behaviour and internal design of a software product of moderate
complexity.
* Able to collaborate with peers to work on a multi-person software project of moderate complexity.
